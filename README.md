# Open Skills

Lightweight skill documentation for common document creation tasks.

## Skills

| Skill | Description |
|-------|-------------|
| [pdf](pdf/SKILL.md) | Create and manipulate PDF documents |
| [xlsx](xlsx/SKILL.md) | Create and edit Excel spreadsheets |
| [docx](docx/SKILL.md) | Create Word documents with tables and formatting |
| [pptx](pptx/SKILL.md) | Create PowerPoint presentations with charts |

## Test Prompts

Use these prompts to test each skill:

### PDF Skill

```
Create a research paper PDF about Einstein's Theory of General Relativity. Include:

1. Title page with paper title, author name, and date
2. Abstract section
3. Introduction explaining the historical context
4. Main sections covering:
   - Special relativity foundations
   - The equivalence principle
   - Curved spacetime and gravity
   - Experimental confirmations
5. Mathematical equations where appropriate
6. Conclusion
7. References section

Use proper academic formatting with headers, page numbers, and citations.
Save to output/relativity-paper.pdf
```

### Excel Skill

```
Create a SaaS financial model spreadsheet with:

1. Revenue sheet:
   - Monthly MRR projections for 24 months
   - Customer count by tier (Starter, Pro, Enterprise)
   - ARPU calculations
   - Churn rate assumptions

2. Expenses sheet:
   - Headcount plan by department
   - Salary costs
   - Operating expenses (hosting, tools, marketing)

3. Summary sheet:
   - Monthly P&L
   - Key metrics (LTV, CAC, LTV:CAC ratio)
   - Charts showing revenue growth and burn rate

Use proper formatting with headers, currency formats, and formulas.
Save to output/saas-financial-model.xlsx
```

### Word Document Skill

```
Create a professional consulting proposal Word document for a digital transformation engagement. Include:

1. Cover page with company logo placeholder, proposal title, client name, and date
2. Executive summary
3. Understanding of client needs section
4. Proposed approach with phases/timeline table
5. Investment summary table with pricing breakdown
6. Team qualifications
7. Terms and conditions

Use proper formatting including headers, professional styling, and well-formatted tables.
Save to output/consulting-proposal.docx
```

### Slides Skill

```
Create a professional pitch deck presentation for "Sprintos" - a fictional project management SaaS startup. Include:

1. Title slide with company name and tagline
2. Problem slide - pain points in project management
3. Solution slide - how Sprintos solves these problems
4. Product features slide with key capabilities
5. Market opportunity slide with a chart showing market size/growth
6. Business model slide
7. Traction slide with a chart showing growth metrics
8. Team slide
9. Financial projections slide with a chart
10. Call to action / investment ask slide

Use modern, clean design with consistent colors. Include charts for the market, traction, and financial slides.
Save to output/sprintos-deck.pptx
```
